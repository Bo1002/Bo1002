## 🚀 Featured Projects
 
### 🔵 Bootloader Development – S32K144
 
> UART-based bootloader on NXP S32K144 with secure firmware upgrade mechanism.
 
- ✅ Designed dual-region flash layout (BOOT / USER APP) with boot mode selection mechanism
- ✅ Developed UART-based firmware upgrade flow: SREC parser, circular queue buffer, and sector-level flash programming driver
- ✅ Implemented CRC checksum validation and secure jump-to-application with full memory boundary protection
---
 
### 🟠 CMSIS Driver Library & ETB Motor Control – S32K144
 
> Full peripheral driver library following CMSIS standard + closed-loop ETB motor control simulator.
 
- ✅ Implemented full peripheral driver library (FlexCAN, FTM, ADC, UART, SPI, I2C) following CMSIS standard with register-level configuration
- ✅ Built closed-loop ETB simulator: ADC (throttle) → PID (10ms LPIT) → FTM PWM (motor) → FTM Quadrature Decoder → FlexCAN TX (500 kbps live RPM)
- ✅ Applied AUTOSAR-inspired three-layer architecture (Application / Driver / Hardware), zero global variable sharing between layers
---
 
### 🟢 Industrial Motor Monitoring & Control – Arduino R4 *(School Project)*
 
> Real-time industrial motor monitoring and control system.
 
- ✅ Designed embedded control board with optical speed sensor and load cell, implementing ADC-based signal conditioning for real-time closed-loop motor control
- ✅ Developed closed-loop frequency regulation algorithm adjusting inverter output based on speed and load feedback
- ✅ Integrated LabVIEW serial interface for real-time data visualization and logging
